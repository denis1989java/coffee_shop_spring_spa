1. В файле root.properties в строке 1 нужно выбрать как будешь хранить заказы. В строка 4-6 примеры как нужно писать свой выбор.
2. Если выбрал файл то в этом же файле нужно выбрать путь куда ты будешь их класть, и соответсвенно создать нужные директории в файловой системе - только папки, сами файлы создавать не нужно.
3. Если выбрал MySQL - запустится файл init.sql - который проверит существует ли база данных coffee_shop, и если нет создаст. Затем выполнит ряд других команд для работы приложения. Так если база существовала, и существовали таблицы с нужными именами - они удалятся и зоздадутся и заполнятся новые.
4. если выбрал Oracle - ничего не произойдет - приложение будет работать с единой - созданной нами БД.
5. Чтобы зайти на страницу админа - нужно зайти по хэшу: #admin, если еще не залогинился - залогиниться (логин и пароль admin).
